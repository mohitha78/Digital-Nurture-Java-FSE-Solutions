# Exercise 2: Implementing the Factory Method Pattern

## Objective

Implement the Factory Method Design Pattern to create different types of documents such as Word, PDF, and Excel.

## Files Included

* Document.java
* WordDocument.java
* PdfDocument.java
* ExcelDocument.java
* DocumentFactory.java
* WordDocumentFactory.java
* PdfDocumentFactory.java
* ExcelDocumentFactory.java
* FactoryMethodTest.java
* FactoryMethodPattern_Output.png

## Description

The Factory Method Pattern delegates the responsibility of object creation to factory classes. Each factory creates a specific type of document.

## Output

The program successfully creates and opens Word, PDF, and Excel documents using their respective factory classes.

## Result

Successfully implemented and tested the Factory Method Design Pattern in Java.
