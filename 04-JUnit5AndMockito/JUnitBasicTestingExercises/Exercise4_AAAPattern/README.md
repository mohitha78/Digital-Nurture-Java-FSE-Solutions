# Exercise 4: Arrange-Act-Assert (AAA) Pattern, Test Fixtures, Setup and Teardown Methods in JUnit

## Objective
Learn to organize JUnit test cases using the Arrange-Act-Assert (AAA) pattern and use setup and teardown methods with `@Before` and `@After`.

## Concepts Used
- Arrange-Act-Assert (AAA) Pattern
- @Before
- @After
- assertEquals()

## Tools Used
- Java 17
- Apache Maven 3.9.16
- JUnit 4.13.2
- Visual Studio Code

## Result
The test executed successfully using the AAA pattern. Setup and teardown methods ran before and after the test, and the project built successfully with no failures or errors.
